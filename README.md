# Ryze Website Redesign

This project is a redesigned marketing website for Get-Ryze.ai built using React and Vite.

## Features
- Modern, clean UI
- Modular and reusable components
- Responsive design
- Clear call-to-actions
- Separate pages using React Router

## Pages
- Home
- Pricing
- About

## Tech Stack
- React
- Vite
- CSS
- React Router
